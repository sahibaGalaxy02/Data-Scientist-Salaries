#📊 Data Scientist Salary Dashboard



🔍 Overview
This Power BI Dashboard provides an interactive analysis of Data Scientist salaries across different regions, experience levels, and industries. It helps users gain insights into salary trends, job market demand, and factors influencing compensation.

📈 Key Features

✔ Salary Trends – Visualization of salary distributions across various experience levels.
✔ Geographical Analysis – Comparison of salaries across different locations.
✔ Industry Insights – Breakdown of salaries based on industries hiring Data Scientists.
✔ Filters & Slicers – Interactive controls to analyze salary data dynamically.

🎯 Purpose

This dashboard is designed to assist aspiring Data Scientists, recruiters, and analysts in understanding salary benchmarks and market trends for better decision-making.
